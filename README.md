# Quick_Service_Restaurant_Inventoy_Forecasting
AI-powered inventory optimization system for quick service restaurants — demand forecasting, waste reduction &amp; supplier recommendations using Python and time series modelling.

Quick service restaurants operate on thin margins where inventory mismanagement directly eats into profitability. Over-ordering perishable ingredients leads to significant waste costs, while under-ordering causes stockouts, lost sales, and expensive emergency purchases.
This project builds an end-to-end inventory optimization pipeline that predicts how much of each ingredient a restaurant should order each week — reducing waste, recovering lost money, and catching hidden supply risks before they become a crisis.
